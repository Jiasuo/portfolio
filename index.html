À propos

Hello! Je m’appelle Jiasuo et j’adore créer toute sorte de choses sur internet. Mon intérêt dans le développement web a débuté en 2017 lorsque j’ai décidé d’apprendre à coder pour créer des trucs cools. J’ai fait mes premiers pas avec Python.

Aujourd’hui, j’ai eu le privilège de travailler dans une agence de com’, une start-up, une grande entreprise, et un studio de design amateur. Ma priorité actuelle est d’aider les start-ups et les PMEs à transmettre leurs valeurs de marque à travers des produits et services digitaux.

Je suis également en-train de créer un tutoriel vidéo qui explique tout ce qu’il y a à savoir pour créer une application web de numérisation de formulaire papier.

Voici quelques technologies avec lesquelles j’ai travaillé récemment :

  • JavaScript (ES6+)
  • Vue.js
  • Node.js
  • TypeScript
  • Python
  • Linux
